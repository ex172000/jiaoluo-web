我们团队成员经历的一个小故事

他原本是在一个小县城的小学读书，毕业了面临升学问题，他的父母希望他在当地的一所比较好的初中上学，但是他了解到在市区有一所更好的私立初中也在招生，而且面向附属的各个县。他觉得这也是一个开眼界的好机会，于是就和同学们一起参加考试并被录取了。这时候他的父母就面临了一个择校难题，到底去哪里？于是他们当时就去咨询了他们小学经验丰富的老师，那老师建议他们不要去市区，因为孩子太小容易“管不住”。

不甘心之前的努力付之东流，他开始联系去了那所学校的学长。学长分享了他在市区学校的正面经历。说这里一切都是全新的，师生都充满活力、积极向上。学校的成绩也很好，过去几年都是大市第一。最重要的是学长还分享了住校生活的体验，觉得他毋需担心。“学校不仅在学业上有严格管理，生活上也有很多支持” 于是他就把这些信息告诉了先前并不了解这所学校的父母让他们意识到其实放他去市区上学不仅不是一个坏的想法，也能培养独立能力，于是就顺从了他的心愿。

得出的结论：1. 许多事情多一个经历多一个角度想法会不一样 2. 找到合适（相似经历）的回答非常重要

角落平台和其他的问答平台或者论坛的区别?


平时有一些问题/困惑你会怎么解决呢？问身边的人? 搜索平台? 上论坛或者问答平台? 我们研究了这几种解决方式发现了以下三个问题：

回答缺乏实践性

  • 广而泛的回答针对性弱
  • 基于回答者自身经历的回答不知道是否试用
  • 过多的信息难以筛选，效率低（这点在论坛尤为明显）

回答不能产生共鸣，难以找到合适自己的回答

问题本身广而泛

我们想做的平台是

可以向指定的经历者群体提问，答题者根据自己的人生感悟进行回答，APP不仅会对所有回答进行数据统计，告诉我不同经历者的看法数据，并且会推荐匹配度最高的回答给提问者/浏览者。

  • 得到共鸣和支持
  • 有实践性的回答
  • 有效地找到适合我的回答

目标是：

迷茫和困惑能够得到有效地解决

亲历者：有过亲身经历并提取出了人生感悟的人

每个人都是经历者，今天他有可能是昨天的你，我们认为每个人在不同阶段的经历都是有价值的，且有意义的，当把不同的经历和不同角度的想法汇总在一起，所得出来的感悟是更为全面的，有深度的
